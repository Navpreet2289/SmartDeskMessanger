# SmartDeskMessanger
This is messanger application which is based on ionic 3.7 for SmartDesk

## Install Environment
NPM: 3.10.10

NVM: v6.10.2

IONIC: 3.7

## Install
You need to install nvm to get up several npm versions on your machine.

After install nvm, please run the following commands.

nvm use v6.10.2

npm install

ionic serve
